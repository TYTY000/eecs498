#transfer_learning 

|         | 数据分布和我原来的数据比较相似时      | 数据分布不同时          |
| :------ | :-------------------- | :--------------- |
| 新的数据量很小 | 直接在原来的模型上面再用一层线性层进行处理 | 比较难调试，需要不断地进行试错  |
| 新的数据量很大 | 通常需要finetune一些层的参数来匹配 | 需要finetune整个模型参数 |
**迁移学习可以降低几倍训练的时间，但是比finetune一个已经训练好的模型，更好的方法是找到更多的新数据进行训练。**
